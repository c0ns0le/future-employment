O*NET 17.0 Database
July 2012 Release
-----------------

The current release of the O*NET database is available online at:

http://www.onetcenter.org/database.html

For information on future releases of the O*NET database, visit
the O*NET Development Database page at:

http://www.onetcenter.org/dev_db.html


New in 17.0
-----------

This release includes new and updated data for 108 O*NET-SOC occupations.
In addition, this release includes new and updated Work Values data for 82
occupations. As of the 17.0 release, all 974 data-level occupations in the
O*NET-SOC 2010 taxonomy have analyst Work Values ratings.


For More Information
--------------------

For comprehensive information on the contents, structure, and
relationship of the database files, and a history of changes in each
database release, please consult the O*NET Data Dictionary 17.0:

http://www.onetcenter.org/dl_files/DataDictionary17_0.pdf
